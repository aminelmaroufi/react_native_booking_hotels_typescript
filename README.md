## Description

A booking hotels mobile app with React Native, Typescript and Redux-saga.

## Installation

```bash
$ npm install
```
for ios 
```bash
cd ios && pod install
``` 

## Running the app

```bash
# android
$ npm run android

# ios
$ npm run ios
```

## Test

```bash
# unit tests
$ npm run test
```
